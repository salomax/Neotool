# Feature Development Workflow

> **Purpose**: Complete feature development workflow from idea to production.

## Overview

This workflow guides feature development from initial idea through implementation to production deployment.

## Steps

1. **Discovery**: Complete feature questionnaire
2. **Form Filling**: Fill feature creation form
3. **Planning**: Generate implementation plan
4. **Implementation**: Build feature
5. **Validation**: Complete checklists and review
6. **Deployment**: Merge and deploy

See [Feature Creation Workflow](../04-templates/feature-creation/workflow.md) for detailed steps.
