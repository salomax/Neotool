# Create Frontend Feature AI Prompt

> **Purpose**: AI prompt template for creating frontend features.

## Usage

Use this prompt when implementing frontend features. Reference the feature form and implementation plan.

## Prompt Template

```
Create a frontend feature for NeoTool following the specification.

## Feature Details

[Feature name and description from feature form]

## Frontend Requirements

[Frontend section from feature form]

## Specification References

- Component Pattern: spec/03-patterns/frontend/component-pattern.md
- Page Pattern: spec/03-patterns/frontend/page-pattern.md
- Hook Pattern: spec/03-patterns/frontend/hook-pattern.md
- GraphQL Pattern: spec/03-patterns/frontend/graphql-pattern.md
- Styling Pattern: spec/03-patterns/frontend/styling-pattern.md

## Implementation Requirements

1. Create components following component-pattern.md
2. Create pages following page-pattern.md
3. Create hooks following hook-pattern.md
4. Create GraphQL operations following graphql-pattern.md
5. Apply styling following styling-pattern.md
6. Add i18n support
7. Create tests

## Deliverables

- Page components
- Feature components
- Custom hooks (if needed)
- GraphQL operations
- i18n strings
- Tests

Generate the complete frontend implementation.
```
